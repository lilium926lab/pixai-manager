# PixAI Prompt & Model Manager

このリポジトリは、PixAI 用のプロンプト、ベースモデル、LoRA モデルを管理するためのシンプルなウェブアプリケーションです。ローカルストレージにデータを保存し、GitHub Pages などの静的ホスティングで動作します。

## 特徴

- **プロンプト管理**: 名前とプロンプトテキストを入力し、検索や削除ができます。
- **ベースモデル管理**: モデル名、タイプ、タグ、商用利用可否を登録し、検索や商用利用のみのフィルター、削除が可能です。
- **LoRA 管理**: モデル名、ベースモデル、タグ、商用利用可否を登録し、検索や商用利用のみのフィルター、削除が可能です。
- **ローカルストレージ保存**: 入力したデータはブラウザのローカルストレージに保存され、ページを再読み込みしても残ります。
- **レスポンシブデザイン**: シンプルで見やすいインターフェイスを提供しています。

## デプロイ方法

1. このリポジトリをクローンまたはダウンロードします。
2. GitHub Pages を使用する場合、`Settings` → `Pages` から `Branch` を `main`/`master` の `root` に設定してください。
3. `index.html` がトップページとして公開されます。

## ファイル構成

- `index.html` – アプリケーションのメイン HTML ファイル。
- `style.css` – レイアウトや色彩を定義するスタイルシート。
- `script.js` – データの追加・削除・検索・フィルタリングなどのロジックを提供する JavaScript。
- `README.md` – この説明書。
